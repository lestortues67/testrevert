Ce dossier appelé 'packages' est un PACKAGE. Il contient un fichier __init__. Ce fichier indique au système que ce dossier est un PACKAGE. 

Un PACKAGE contient des fichiers python appelés MODULE. 

Les MODULES contiennent des définitions diverses telles que classes, fonctions, etc ...

Ils aident à diviser un projet important en plusieurs fichiers de code source.
